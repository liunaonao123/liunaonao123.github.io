AJAX

Ajax简介

Ajax(Asynchronous JavaScriptandXML)并不是一种全新的技术，而是一种使用现有标准的新算法，也就是说他不是一门语言，是一种方法，由JavaScript、XML、CSS等几种现有技术整合而成。

Ajax工作流程

用户界面触发事件调用JavaScript，通过Ajax引擎——XMLHttpRequest对象异步发送请求到服务器，服务器返回数据，然后利用返回的数据使用DOM和CSS技术局部更新用户界面。

XmlHttpRequest对象

XMLHtpRequest对象可以在不刷新页面的情况下向服务器发送异步请求(异步传输是面向字符的传输，它的单位是字符)，并接收服务器端的响应结果，从而实现局部更新当前页面的功能。XmlHttpRequest对象不仅可以接收xml格式的数据，还可以接收Json或Html等格式的文档数据。

XmlHttpRequest对象的常用方法

方法名称 说明
open(String post,String url …) 用于创建一个新的HTTP请求
send(String data) 发送请求到服务器端
abort() 取消当前请求
setRequestHeader(String header,String value) 单独设置请求的某个HTTP头信息
getResponseHeader(String header) 从响应中获取指定的HTTP头信息
getAllResponseHeaders 获取响应的所有头信息

XmlHttpRequest对象的常用属性

属性名称 说明
onreadystatechange 设置回调函数
readyState 返回请求的当前状态
常用值
0——未初始化
1——开始发送请求
2——请求发送完成
3——开始读取响应
4——读取响应结束
status 返回当前请求的HTTP状态码
常用值
200——正确返回
404——找不到访问对象
statusText 返回当前请求的响应行状态
resopnseTest 以文本形式获取响应值
resopnseXML 已XML形式获取响应值，并且解析成DOM对象返回

使用Ajax发送GET请求

function validate(){
    // 提示信息
    var msg = document.getElementById("msg");    
    // 获取参数
    var username =document.getElementById("username").value; 
    // 请求验证的地址，及GET参数
    var url = "validate_username_servlet?username="+username;
    // 创建Ajax引擎    
    var xhr = new XMLHttpRequest();    
    //请求状态更改时，执行方法
    xhr.onreadystatechange = function(){
         var readyState = xhr.readyState;    // 请求状态
         var status = xhr.status;        // HTTP状态码
         //请求成功，并没有错误
         if(readyState==4 && status==200){
            // 服务器响应结果，输出的结果
            var result = xhr.responseText;
            if(result=="true"){
                msg.innerHTML = "邮箱已存在！";
            }else{
                msg.innerHTML = "邮箱可以使用！";
            }
        }
    };
        xhr.open("GET", url, true);    // 设置请求方式及地址
        xhr.send(null);                // 发送请求
}
